# База знаний по Электронному бюджету

Проект использует [MkDocs](https://www.mkdocs.org/) с темой [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) и наобором расширений [PyMdown](https://facelessuser.github.io/pymdown-extensions/) для преобразования файлов Markdown, расположенных в папке `docs/`, в корне этого проекта, в статический сайт, который может быть размещён на таком ресурсе, как [GitVerse Pages](https://gitverse.ru/docs/pages/overview/) или [GitHub Pages](https://docs.github.com/ru/pages).
